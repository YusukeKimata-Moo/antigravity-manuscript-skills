# 学術英語表現ガイド

## 動詞・述語の選択

| 日本語                   | 推奨英訳                               | 避けるべき訳             | 理由                                            |
| ------------------------ | -------------------------------------- | ------------------------ | ----------------------------------------------- |
| 〜を明らかにした         | revealed / demonstrated / showed       | made clear               | 口語的で学術論文に不適                          |
| 〜が示された             | was shown / was demonstrated           | was indicated            | indicatedは間接的証拠のニュアンスが強すぎる     |
| 〜が示唆された           | suggested / implied                    | was suggested to be      | 受動態＋to不定詞は冗長                          |
| 〜が認められた           | was observed / was detected            | was recognized           | recognizedは知覚・認知の意味で実験観察に不適    |
| 〜と一致する             | is consistent with / corroborates      | matches                  | matchesは完全一致のニュアンスで科学的議論に不適 |
| 〜に重要/必須である      | is essential for / is required for     | is important for         | important forは主張が弱い（弱い文脈なら許容）   |
| 〜の可能性がある         | may / might / could                    | there is a possibility   | 冗長な名詞化表現                                |
| 〜とは異なる             | differs from / is distinct from        | is different from        | 口語的で学術文体に合わない                      |
| 有意差は検出されなかった | No significant difference was detected | There was no significant | There構文は主語を曖昧にし弱い表現になる         |
| 〜を目的とした           | aimed to / sought to                   | purposed to              | purposedは非標準的用法                          |

## 主張の強度（hedging）

| 強度 | 英語表現                              | 使用場面                   |
| ---- | ------------------------------------- | -------------------------- |
| 強   | demonstrate, establish, determine     | 直接的な実験証拠がある場合 |
| 中   | show, reveal, indicate                | 明確なデータに基づく場合   |
| 弱   | suggest, imply, raise the possibility | 間接的証拠や推測の場合     |
| 最弱 | may, might, could, is conceivable     | 仮説レベルの議論           |

## 接続・展開表現

| 機能   | 推奨表現                                                         |
| ------ | ---------------------------------------------------------------- |
| 追加   | Furthermore, Moreover, In addition, Additionally                 |
| 対比   | However, In contrast, By contrast, Conversely, On the other hand |
| 因果   | Therefore, Thus, Consequently, Accordingly                       |
| 具体化 | Specifically, In particular, Notably                             |
| 譲歩   | Although, Despite, Notwithstanding                               |
| 要約   | Taken together, Collectively, Overall                            |

> **注意**: 同一接続詞の連続使用を避ける。特に "However" と "In addition" の多用に注意。
