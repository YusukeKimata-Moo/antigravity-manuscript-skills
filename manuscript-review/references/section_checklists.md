# セクション別チェックリスト

## Introduction

- [ ] **パラグラフ構造**: 一般的背景 → 先行研究 → 未解決の問題 → 本研究の目的・アプローチ → 結果の要約 の順で構成されているか
  - ✅ 各パラグラフが1つの論点に集中し、次へ自然につながる
  - ❌ 背景と先行研究が混在し、未解決の問題が唐突に出現する
- [ ] **トピック順序**: Resultsの展開順と一致しているか
  - ✅ Introで「極性形成→液胞動態→細胞分裂」の順に触れ、Resultsも同順
  - ❌ IntroではA→B→Cの順だが、ResultsではB→A→Cの順
- [ ] **最終パラグラフの役割**:
  - 研究目的・手法・対象が明確に述べられているか
  - 結果の要約は**抽象的な大枠にとどめ**、Resultsの具体的内容を先取りしすぎていないか
  - 意義づけがDiscussionの結論と**重複していないか**（Introは事実ベースに留め、解釈や考察はDiscussionに委ねる）
  - ✅ 「本研究ではライブイメージングにより液胞動態を解析し、極性形成との関連を明らかにした」
  - ❌ 「液胞の再編成が極性軸決定に必須であることを実証し、新しい発生モデルを提唱した」（Discussionの結論を先取り）

## Results

- [ ] **セクション見出し**: 各セクションの内容を端的に表す見出しがついているか
  - ✅ 「液胞は発芽開始時に急速な形態変化を示す」（結果を要約する能動的見出し）
  - ❌ 「液胞の観察」（内容不明な中立的見出し）
- [ ] **手法の切り替え**: 異なる手法（例: 固定標本観察→ライブイメージング）の切り替え時に接続語で明示しているか
  - ✅ 「固定標本での形態観察に加え、ライブイメージングにより動態を解析した」
  - ❌ 手法が変わったことに言及なく次の結果を提示
- [ ] **定量データの提示**: 定性的記述だけでなく、定量解析の結果（有意差の有無等）が示されているか
- [ ] **対照・検証**: 独立した手法やマーカーによる再現性の確認への言及があるか
- [ ] **先行研究との比較**: 各セクション末尾で関連する先行研究との一致/相違を簡潔に述べているか
- [ ] **Figure参照の一貫性**: Figure番号が正しく、同一番号が意図せず重複していないか

## Discussion

- [ ] **パラグラフ構成**: 方法論的貢献 → 個別知見の考察（Resultsの順に対応） → 知見の総合・機能的含意 → 機構的考察 → 展望 の流れが自然か
- [ ] **Resultsとの対応**: Resultsで示したすべての主要知見がDiscussionで議論されているか
  - ✅ Resultsの各サブセクションに対応するDiscussionパラグラフがある
  - ❌ Resultsで示した重要な知見がDiscussionで言及されていない
- [ ] **論理の飛躍チェック**: Discussionで主張する内容がResultsのデータで裏付けられているか。データに示されていない事実を前提にしている場合、先行研究の引用で根拠を明示する
- [ ] **パラグラフ間の重複**: 同じ結論が複数パラグラフで繰り返されていないか。重複がある場合：
  - 一方を**問題提起**に留め、他方で**結論・展望**を述べる
  - 具体的な今後の実験計画は最も関連の深いパラグラフに集約する
- [ ] **新規情報の導入**: Introで触れていない概念や生物種をDiscussionで引用する場合、文脈の説明やデータの限界を明記する
- [ ] **最終パラグラフ**: 研究全体の意義 → 手法の汎用性 → 今後の展望、で締める

## Methods / Supplemental Methods

- [ ] **整合性**: Methodsの記述がSupplemental Methodsの詳細と矛盾していないか
- [ ] **条件の正確性**: 実験条件・試薬名・機器設定・解析手法がセクション間で一致しているか
