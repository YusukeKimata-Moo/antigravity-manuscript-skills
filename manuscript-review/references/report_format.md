# 分析レポートフォーマット

セクション間の整合性チェック結果は以下の形式で報告する。

## テンプレート

```markdown
### ✅ 問題なし

| 確認項目             | 判定 |
| -------------------- | ---- |
| トピック順序の整合性 | ✅   |
| 用語の一貫性         | ✅   |
| ...                  | ...  |

### ⚠️ 指摘事項（N点）

#### 1. 【カテゴリ】問題の要約

- **箇所**: ファイル名・行番号
- **現在の記述**: 該当箇所の引用
- **問題点**: 具体的な説明
- **提案**: 最小限の修正案
```
